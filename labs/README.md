# Лабораторні роботи

У папці збережено два типи матеріалів:

- `Лабораторна робота 1.pdf` ... `Лабораторна робота 5.pdf` - вихідні умови завдань.
- `lab1_report.md` ... `lab5_report.md` - підготовлені звіти за виконаними роботами.
